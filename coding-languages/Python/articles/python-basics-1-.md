## 文字を出してみよう！

まずは、文字を出力してみましょう！
Pythonで文字を出力するには<code>print()</code>関数を使うとできます。

例えば、こんにちは!と出力したいときは<br>
<code>
print("こんにちは！")
</code><br>
と記入することでこんにちは!と出力することができます。
なので、Hello!と出力したい場合は<br>
<code>
print("Hello!")
</code><br>
と記入することでHello!と出力することができます。
